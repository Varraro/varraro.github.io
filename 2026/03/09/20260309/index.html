事件

3.9 日 miku！

伊朗与美以之间开始战斗，结果我的事件合约被制裁了，模式直接变了（悲）
OpenClaw 开始被大规模推广了，有点当年充话费送手机的感觉（）

信息

  1. 把老登的东西用 Trae 加上 ACE 的 MCP ，总算是做出来了，后面整 PPT 就行了

  2. 报了国企面试笔试辅导，秋招接好运！

  3. OpenClaw 把兄弟的股票框架里面的宏观层做出来了，很舒服🥰，接下来继续完善

灵感

  1. 事件合约总是会把胜率磨平到50%，无论做出什么努力，所以这次换了一个思路，目前胜率奇高无比，在被磨平之前想办法实盘大赚特赚吧，说实在的现在对于这种基于K线的信任度很低了
  2. 打算看看那个强化学习做的合约，其实强化学习本质上也是乱碰，光分析K线这样子真的有什么规律嘛，我还是更相信接近本质的东西
  3. 之前得到多空比的一些想法，想要让 Openclaw 开动起来帮我实现了，现在Skills真的很多，也不用愁数据这一块的

系统

  1. 用 TV 构建了新的事件合约策略，但是我认为还是需要进行优化，当然现在在那些个标的上表现的非常好，实盘一周试试。上周的实盘因为币安APP更新全失败了，这次重新构建了，币安应该公开事件合约 api 接口的 y1s1

  2. 继续完善兄弟的中观和微观的思路，应该这一周就有结果了，毕竟数据这一块儿彻底是搞定了。

碎碎念

  1. 本来想画只 miku 庆祝一下 39 的，但是拿起笔来真的好生疏，设备也变了，华为的笔不如三星的Spen，唔慢慢复健吧

  2. 看到面试笔试的课快晕过去了，真的好多（）

  3. 《葬送的芙莉莲 s1》 看完啦！好久没看到这么清新脱俗的公路片了，接下来开始看《奇巧计程车》

  4. 想起来上周的周刊没写，唔，只能说太忙了（）
